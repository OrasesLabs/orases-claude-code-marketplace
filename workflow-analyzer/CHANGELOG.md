# Changelog

All notable changes to the Workflow Analyzer plugin.

## [0.1.0] - 2025-11-14

### Added
- Session analyzer skill - Discover and list Claude Code sessions
- Transcript condenser skill - Compress large session transcripts
- Workflow analyzer agent - Comprehensive session analysis
- Event hooks - `pre_compact.py`, `session_end.py`, `subagent_stop.py`
- Slash command - `/analyze-workflow` for workflow analysis
- Documentation - README, INSTALL, QUICKSTART, CHANGELOG
- Stub implementations for future workflow analysis features
